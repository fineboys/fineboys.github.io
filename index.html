Markdown-语法的学习

markdown

Markdown 是一种轻量级标记语言，它允许人们使用易读易写 的纯文本格式编写文档。

Markdown 语言在 2004 由约翰·格鲁伯（John Gruber）创 建。

Markdown 能被使用来撰写电子书，如：Gitbook。当前许多 网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上 发表消息。例如：GitHub、简书、reddit、Diaspora、Stack Exchange、OpenStreetMap 、SourceForge等。

markdown 标题

markdown标题有两种格式

使用=和 - 标记一级标题和二级标题

使用#号标记可以表示1到6级，一个#号代表一级，二级对应 两个#号。以此类推

markdown段落

Markdown 段落没有特殊的格式，直接编写文字就好，段落 的换行是使用两个以上空格加上回车。

markdown字体

斜体文本
例子：你很帅
粗体文本
例子：你很大
粗斜体文本
例子：**你又帅又大**

分割线

你可以在一行中用三个以上的星号、减号、底线来建立一个分 隔线，行内不能有其他东西。你也可以在星号或是减号中间插 入空格。

删除线

如果段落上的文字要添加删除线，只需要在文字的两端加上两 个波浪线即可

下划线

脚注

脚注是对文本的补充说明。

markdown列表

Markdown 支持有序列表和无序列表。

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些 标记后面要添加一个空格，然后再填写内容

有序列表使用数字并加上 . 号来表示

有序列表使用数字并加上 . 号来表示

markdown区块

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧 跟一个空格符号

例子：

>shuai
>xuexi
>jiuwanshi

shuai

xuexi

jiuwanshi

markdown代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起 来（`）

代码区块使用 4 个空格或者一个制表符（Tab 键）

你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指 定）

markdown链接

链接使用方法有两种

1、链接名称+链接地址 链接名称

2、链接地址 <链接地址>

Linux的权限管理
chown：改变文件拥有者
chown 用户名 文件/目录
chown 用户名：用户组：文件/目录
若是整个目次下的都改，则加-R参数用于递归更改。

chgrp：转变文件所属用户组
chgrp 用户组 文件名
递归更改文件用户组，+-R就是整个目录下的文件用户组都会更改

chmod：变动文件属性
chmod -R 123 文件或者目录
进行递归更改，同目录下所有文件都会变更

对于服务器需要配置特定账户的看法
我们都知道linux是多用户多任务系统，大多数用作服务器，一台服务器会有很多人登陆管理。
此时可以通过用户组来设置权限就非常方便了，比如某个用户组只能看到某几个文件夹，其他的这个用户组不可见。如果只能对单个用户配置权限设置，既不方便，也不安全。
